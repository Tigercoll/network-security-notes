## 学习内容

### 1️⃣ 安全策略框架

```yaml
# security_policy.yaml - 统一安全策略配置

# 网络访问控制策略
network_access:
  default_policy: "deny"
  allow_list:
    - protocol: "ssh"
      source: "10.0.0.0/8"
      destination: "any"
      port: 22
      condition: "mfa_verified"
    
    - protocol: "http"
      source: "any"
      destination: "dmz"
      port: 80
      condition: "waf_enabled"
```

## 变更内容

### 变更前
```bash
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
```

### 变更后
```bash
iptables -A INPUT -p tcp -s 10.0.0.0/8 --dport 22 -j ACCEPT
iptables -A INPUT -p tcp --dport 22 -m limit --limit 3/min -j ACCEPT
```

## 回滚计划

```bash
iptables -D INPUT -p tcp -s 10.0.0.0/8 --dport 22 -j ACCEPT
iptables -D INPUT -p tcp --dport 22 -m limit --limit 3/min -j ACCEPT
```
